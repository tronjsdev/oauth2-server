## oauth2-server

This is my personal nodejs oauth2 server project

#### Usage

Development:
```
npm run dev
```
\
Debugging:
```
npm run debug
```

\
Build production:
```
npm run build
```

\
Run production (using `pm2`):
```
npm start
```
